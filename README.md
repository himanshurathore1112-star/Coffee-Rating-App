# Coffee Rating Application

A web application where users can vote for their favorite coffee.

## Features
- Display coffee list
- Vote for coffee
- Store votes in SQLite database
- Real-time vote updates

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- SQLite3

## Run Project

npm install

node server.js

Open:
http://localhost:5000
